# showcase

Synthetic reference pack. Not for production — every action here is a
documentation artefact for pack authors. Read alongside
[`docs/action-packs.md`](../../../docs/action-packs.md).

| Action                          | Demonstrates                                               |
| ------------------------------- | ---------------------------------------------------------- |
| `showcase.every_arg_type`       | One arg of every supported type with a typical validation. |
| `showcase.path_validation`      | `allowed_prefixes`, `denied_paths`, `denied_prefixes`.     |
| `showcase.json_output`          | `parser: json`, `parser_required`, per-action redaction.   |
| `showcase.opts_envelope`        | `timeout_min/max`, `max_stdout_bytes_min/max`, clamping.   |
| `showcase.script_action`        | `kind: script`, interpreter, script SHA-256 journaling.    |
